# 🛒 أسواق خيرات السبتي

## طريقة التشغيل

---

## 🖥️ على اللابتوب (أول مرة فقط)

### الخطوة 1 — حمّل Node.js
- اذهب لـ: https://nodejs.org
- حمّل النسخة **LTS** (الخضراء)
- نزّلها وثبّتها عادي

### الخطوة 2 — افتح Terminal (موجه الأوامر)
- **Windows**: ابحث عن `cmd` أو `PowerShell`
- **Mac**: ابحث عن `Terminal`

### الخطوة 3 — روح لمجلد المشروع
```
cd khayraat-app
```

### الخطوة 4 — ثبّت المكتبات
```
npm install
```
(انتظر دقيقتين)

### الخطوة 5 — شغّل البرنامج
```
npm run dev
```

### الخطوة 6 — افتح المتصفح
اذهب لـ: **http://localhost:3000**

---

## 🌐 رفع على النت (Netlify) — مجاني

### الخطوة 1 — ابنِ المشروع
```
npm run build
```

### الخطوة 2 — إنشاء حساب Netlify
- اذهب لـ: https://netlify.com
- سجّل بحساب Google أو GitHub

### الخطوة 3 — ارفع المجلد
- في Netlify اضغط **"Add new site"**
- اختر **"Deploy manually"**
- اسحب مجلد **`dist`** وأفلته على الصفحة

### الخطوة 4 — احصل على رابطك!
Netlify يعطيك رابط مثل:
**`https://khayraat-sabti.netlify.app`**

---

## 📱 على الآيفون (بدون تنزيل)

### بعد رفعه على Netlify:

1. افتح **Safari** على الآيفون
2. اذهب للرابط اللي حصلت عليه
3. اضغط **Share** (زر المشاركة ⬆️)
4. اختر **"Add to Home Screen"**
5. اضغط **Add**

الآن عندك أيقونة على الشاشة الرئيسية تشتغل مثل تطبيق! 🎉
